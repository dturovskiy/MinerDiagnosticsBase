# Шаблон для надсилання проблеми

## Що сталося
Коротко опиши проблему.

Приклад:
> Герой не починає лізти по драбині, якщо підійти зліва і затиснути вверх.

## Кроки відтворення
1. Запустити MainScene
2. Підійти до драбини зліва
3. Натиснути і тримати Up
4. Герой лишається в walk state

## Очікувана поведінка
Герой має перейти в climb state.

## Фактична поведінка
State не змінюється, velocity по Y лишається 0.

  - героя
  - Rigidbody2D
  - колайдера драбини
  - компонентів, що відповідають за state / ladder
