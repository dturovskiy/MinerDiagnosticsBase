# Нотатки по інтеграції у Miner_Unity

У репозиторії `dturovskiy/Miner_Unity` є структура `Assets/Scripts/*`, а в README вказана версія Unity 6000.3.10f1. Це означає, що базу з цього архіву логічно вставляти в `Assets/Scripts/Diagnostics` без окремого пакеджингу. Далі можна поступово під'єднати її до існуючих систем руху та сцени `MainScene`.

Рекомендований порядок:

1. Підключити сам модуль `Diagnostics`
2. Поставити базовий snapshot provider на героя
3. Почати логувати тільки переходи станів
4. Потім додати логування ladder-подій
5. Після цього вже розширювати `dataJson` причинами, таймерами і блокуваннями
